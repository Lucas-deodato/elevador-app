# Elevator App

Aplicação mobile desenvolvida com Expo.

## Objetivo

- Este projeto foi desenvolvido como atividade acadêmica com foco em:
- Construção de interfaces mobile responsivas
- Gerenciamento de estado com React Hooks
- Implementação de atualizações OTA (Over-The-Air)
- Uso de ferramentas modernas de build e deploy

## Funcionalidades

- Lista de comunicados (ScrollView)
- Alternância de tema claro/escuro (useState)
- Categorias de notificações (Manutenção, Segurança, etc.)
- Atualizações OTA com EAS Update
- Interface moderna com cards e hierarquia visual

## Tecnologias

- React Native
- Expo
- EAS Update
- TypeScript
